\section{Fit Values and Errors}
Fitted parameter values and parameter errors for the primary fit 
function. 

\begin{enumerate}
    \item Z Position Fit Error $<$ 0.3 mm
    \item $\phi$ Position Fit Error $<$ 0.03 deg
    \item Reduced $\chi ^{2} \; <$ 100
\end{enumerate}
